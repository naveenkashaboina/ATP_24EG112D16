# Week 2 Assignments (Day 3–5)

Objective  
To strengthen JavaScript skills by applying array methods, modular programming, and Express.js for building real-world applications.

Work Description  
Day 3  
- Practiced array methods (filter, map, reduce, find, findIndex) in scenarios like shopping cart, student dashboard, payroll, movies, banking, and temperature analysis.  
- Built a Task Management System using ES6 modules (task.js, validator.js).  
- Designed a modular E-Commerce Shopping Cart System with product.js, cart.js, discount.js.

Day 4  
- Practiced spread operator and structuredClone for arrays and objects.  
- Implemented rest parameters in functions.  
- Built a Library Book Management System using classes with borrow, return, info, and long book checks.  
- Implemented OTP Countdown Simulator with setInterval and Exam Portal Simulator with setTimeout.  
- Practiced object cloning with spread operator.

Day 5  
- Built a mini Express.js REST API for product management.  
- Implemented CRUD operations in productapi.js.  
- Tested endpoints using req.http (GET, POST, PUT, DELETE).  
- Created server.js to set up Express server with JSON middleware.

Outcome  
- Strengthened knowledge of array methods and their applications.  
- Learned modular programming with ES6 import/export.  
- Practiced object cloning, rest parameters, and classes.  
- Understood timers for asynchronous operations.  
- Built and tested REST APIs using Express.js.  
- Improved ability to solve real-world problems with JavaScript.
