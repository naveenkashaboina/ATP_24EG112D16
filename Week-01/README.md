# Week 1 Assignments

Objective  
To understand basic JavaScript concepts.

Work Description  
- Implemented programs covering variables and data types, conditional statements, loops, and functions.  
- Practiced array operations such as finding largest/smallest numbers, summing elements, and searching.  
- Completed employee management tasks: insert, remove, update employee records.  
- Completed assignments from Day 1 and Day 2.

Outcome  
- Developed fundamental knowledge of JavaScript syntax.  
- Improved basic problem-solving skills.  
- Learned to manipulate arrays and objects with functions.
