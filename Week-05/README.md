# Week 5 Assignments (Day 12–14)

Objective  
To practice building styled web pages and React applications using HTML, CSS, Tailwind CSS, and React components.

Work Description  

Day 12  
- Created card components with flexbox layout.  
- Styled cards with background color, box shadow, border radius, and centered content.  
- Built tables with merged cells using `colspan` and `rowspan`.  
- Designed a marks table with practical and theory marks for multiple students.  
- Applied CSS for borders, spacing, alternating row styles, and rounded corners.

Day 13  
- Built a landing page layout with top, bottom, and footer sections.  
- Top section included heading, paragraph, and button.  
- Bottom section displayed multiple cards with headings, text, and buttons.  
- Footer contained company information.  
- Created a City Guide page with navigation bar, main section, and footer sections (Welcome, Great Food, How to Get Here).  
- Practiced Tailwind CSS for responsive design, grid layouts, and text styling.  
- Applied flexbox and spacing for layout consistency.

Day 14  
- Assignment 1: Product Listing  
  - Defined a list of products in `App` component.  
  - Rendered products using `map()` and passed data to a reusable `Product` component.  
  - Displayed product details such as title, category, price, and rating.  
  - Styled product cards with Tailwind CSS.  
- Assignment 2: User Management Layout  
  - Built `App` component with `Nav`, `UserList`, and `Footer`.  
  - `Nav` component: navigation bar with logo and menu items.  
  - `UserList` component: rendered a grid of user cards using `User` component.  
  - `User` component: displayed user image, name, and email.  
  - `Footer` component: contained multiple sections with links.  
  - Applied Tailwind CSS for layout, spacing, and styling.

Outcome  
- Learned how to design card layouts and structured tables with HTML and CSS.  
- Practiced flexbox and responsive design for arranging content.  
- Strengthened knowledge of Tailwind CSS for styling and grid layouts.  
- Built React applications with reusable components and props.  
- Gained experience in rendering lists dynamically with `map()` in React.  
- Improved ability to structure applications with navigation, content, and footer sections.
