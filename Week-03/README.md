# Week 3 - Day 7 Assignments

Objective  
To build and test a modular Express.js Product API with full CRUD functionality.

Work Description  
- Implemented Product API (`productapi.js`) using Express Router.  
- Added endpoints:  
  - GET /products → Read all products  
  - GET /products/:brand → Read product details by brand  
  - POST /products → Create a new product  
  - PUT /products → Update product by productId  
  - DELETE /products/:id → Delete product by productId  
- Created HTTP request file (`req.http`) to test API endpoints:  
  - POST → Add new product (Laptop, brand DELL)  
  - GET → Fetch all products  
  - GET → Fetch products by brand name (DELL)  
  - PUT → Update product details (brand changed to HP)  
  - DELETE → Remove product by ID  

Outcome  
- Practiced building modular APIs using Express Router.  
- Strengthened understanding of CRUD operations in RESTful services.  
- Learned how to test APIs using .http request files.  
- Improved backend development skills with Express.js.
