# Week 4 Assignments

Objective  
To practice building basic web pages using HTML and CSS, and understand the structure of documents, lists, links, images, and styling.

Work Description  
- Created a basic HTML page with headings, paragraphs, links, and unordered lists.  
- Implemented a nested unordered list to represent categories like Fruits, Vegetables, Cereals, and Nuts.  
- Built a profile page "Pochi the Cat" with headings, image, introduction, profile details, and links.  
- Applied CSS styling to control image display and alignment (float, block display).  
- Practiced combining HTML and CSS for simple webpage layouts.

Outcome  
- Learned how to structure content in HTML using headings, paragraphs, lists, and links.  
- Practiced embedding images and applying CSS styles.  
- Understood how to create nested lists and organize information.  
- Gained confidence in building simple static web pages with HTML and CSS.
